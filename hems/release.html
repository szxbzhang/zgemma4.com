Download IPK ~26 KB

Method 1 — USB Install (Easiest)

1. Download the IPK file above to your computer
2. Copy it to a USB stick
3. Plug the USB into your ZGEMMA
4. On your ZGEMMA go to: Menu → Plugin Browser → Install from USB
5. Select the IPK file and confirm
6. Restart Enigma2 when prompted

Method 2 — One-Line Install via SSH

wget -qO /tmp/hems.ipk https://www.zgemma4.com/hems/enigma2-plugin-extensions-hems_1.1.0_all.ipk && opkg install /tmp/hems.ipk && systemctl restart enigma2
