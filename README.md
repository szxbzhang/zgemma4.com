# zgemma4.com

Community resource website for the **ZGemma 4** Linux-based satellite & cable TV receiver.

## Structure

```
zgemma4.com/
├── index.html        # Main website
├── css/
│   └── style.css     # Styles (dark theme, responsive)
├── js/
│   └── main.js       # Interactive behaviour
└── README.md
```

## Pages & Sections

- **Hero** — introduction with device preview
- **Features** — dual tuner, IPTV, PVR, plugins, WiFi, CI+ slot
- **Specifications** — full technical specs table
- **Firmware Downloads** — OpenPLi, OpenATV, OpenVIX, Open Blackhole with flash guide
- **Support** — quick-start, satellite setup, troubleshooting, plugins, apps
- **Community** — forum links, GitHub, wiki, community statistics

## Development

Open `index.html` directly in a browser — no build step required.
